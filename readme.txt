【バグ】
内容
エネミー本体を2体倒すとバグ発生

原因
CountdownTimer.csのUpdateでifによる表示/非表示の指示が問題と考えられるが、試行錯誤しても根本的解決に繋げられていない。エネミー本体のHPを制限時間内で絶対に倒せない数値にすることでバグ発生を止められるが、このゲームはエネミー本体も倒すことで獲得ポイントの上限突破の可能性の鍵を担っているのでその方法を避けている

現在の対処
2体分のエネミー本体のHPをバグ確認時の10倍の数値にしており、時間内で倒すことができる量でバグ遭遇率低下の方向で判断した。
2体のうち1体倒した場合はバグは起きないが、2体とも倒してしまうとバグが起きてしまうので、今後はしっかりと根本的な対処がされた修正版を完成させられるようにする。
