◆内容解説
・Package.zipはUnityゲームのデータをパッケージ化したフォルダです。
解凍後、ShootingGame.unitypackageをクリック頂くと内容を閲覧できます。
(Unityバーション：2020.3.32f1)
手段がGithubのみとあったので、Export Packageによる方法になり、お手数お掛けしますがよろしくお願いします。
・Script.zipはShootingGameのプログラミングコードをまとめたフォルダです。
